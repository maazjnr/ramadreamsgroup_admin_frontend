# Ramadreams Admin Frontend

React + Vite admin app for managing property content.

## Features
- Admin login
- Protected dashboard routes
- Property create/edit/delete
- Image/video upload preview
- Status and search filtering

## Quick Start
1. Install dependencies:
   `npm install`
2. Create env file:
   `cp .env.example .env`
3. Set API base URL in `.env`:
   `VITE_API_BASE_URL=http://localhost:5000/api/v1`
4. Start dev server:
   `npm run dev`

Default local URL: `http://localhost:5174`

## Production Subdomain Example
- Admin frontend subdomain: `admin.ramadreamsgroup.com`
- Backend API subdomain: `api-admin.ramadreamsgroup.com`
- Set environment:
  `VITE_API_BASE_URL=https://api-admin.ramadreamsgroup.com/api/v1`
